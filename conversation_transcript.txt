# Grok Help

**Grok Build** is a terminal AI coding assistant from xAI. It can read and edit your
codebase, run shell commands, search the web, manage tasks, and more.

## Quick start

| Action | How |
|--------|-----|
| Start Grok | grok |
| Update | grok update |
| Check version | grok --version |
| Authenticate | Browser login on first launch, or set XAI_API_KEY |
| Send a message | Type in the prompt and press Enter |

## Common slash commands

Type / in the prompt for autocomplete:

| Command | What it does |
|---------|----------------|
| /new | Start a fresh session |
| /resume | Load a previous session |
| /compact | Compress history to free context |
| /context | Show context window usage |
| /rewind | Undo conversation to an earlier turn |
| /copy | Copy the last response |
| /export | Export the conversation |
| /quit | Exit Grok |

Skills (like /check-work, /review, /design) also show up as slash commands.

## What I can help with

1. Getting started — install, first launch, basic use
2. Authentication — browser login, API keys, OIDC
3. Keyboard shortcuts — full key bindings
4. Configuration — config.toml, env vars
5. MCP servers — connect tools (GitHub, Linear, etc.)
6. Skills — create custom workflows
7. Theming — appearance
8. Project rules — AGENTS.md
9. Plan mode — design before coding
10. Subagents — parallel specialized agents
11. Background tasks — long-running jobs
12. Permissions & safety — what Grok can/can't do

## Example prompts

- "How do I set up an MCP server?"
- "What keyboard shortcuts are there?"
- "Help me configure authentication with an API key"
- "Create a skill for running my tests"
- Or just describe a coding task
